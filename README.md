<b>Introduction:</b> The framework for a locally hosted website that allows the user to <br/>look through cities in the contiguous US and decide if they are good travel destinations.<br/><br/>
Please ensure project is up to date before pushing any code to avoid merge conflicts.<br/> If unsure of a change notify everyone beforehand.<br/><br/>

  https://www.ncei.noaa.gov/stormevents/ftp.jsp  (2000-2025)<br/>
  https://open-meteo.com/<br/>
  https://simplemaps.com/data/us-cities<br/>
<b>Libraries Used:</b><br/>
  https://crowcpp.org/master/<br/>
  https://github.com/libcpr/cpr<br/>
  <br/>
  <b>Running the Code:</b> A zip file is included within this repository, <br/>to run the code and start the webapp simply open the .bat (Windows) or .command (Mac) file. <br/>If for some reason this does not work, you can run it through the CLion IDE by just clicking run after opening the files locally.
